# Sistem-Deteksi-Insomnia-Berbasis-Arduino-UNO-dan-Algoritma-KNN-Berdasarkan-Parameter-HRV
Sistem deteksi insomnia real-time berbasis Internet of Medical Things (IoMT) menggunakan Arduino UNO, sensor ECG AD8232, dan algoritma K-Nearest Neighbor (KNN) dengan analisis Heart Rate Variability (HRV) untuk klasifikasi kondisi tidur normal dan insomnia secara low-cost, portable, dan real-time.
